v1.5.19 Release Notes - April 17, 2026
========================================

v1.5.19 updates the Go version to address the following security
vulnerabilities:

- CVE-2026-33810
- CVE-2026-32283
- CVE-2026-32281
- CVE-2026-32280

Dependencies
------------

Fabric CA v1.5.19 has been tested with the following dependencies:
- Go 1.26.2
- Ubuntu 22.04 (for Docker images)
- Databases
    - PostgreSQL 17
    - MariaDB 11
