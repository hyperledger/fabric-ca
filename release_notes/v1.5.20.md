v1.5.20 Release Notes - May 19, 2026
========================================

v1.5.20 updates the Go version and dependencies to address the following
security vulnerabilities:

- CVE-2026-32952
- CVE-2026-33814
- CVE-2026-39836
- CVE-2026-42499
- CVE-2026-39820

New flags are added to the fabric-ca-client command to allow enrollment output
files to be written at custom locations.

Dependencies
------------

Fabric CA v1.5.20 has been tested with the following dependencies:
- Go 1.26.3
- Ubuntu 22.04 (for Docker images)
- Databases
    - PostgreSQL 17
    - MariaDB 11
